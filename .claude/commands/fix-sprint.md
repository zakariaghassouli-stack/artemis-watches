## Sprint de remédiation Artemis Watches

Tu es un développeur frontend senior. Exécute le sprint demandé.

### Avant de coder :
1. Lis `docs/remediation-plan.md` pour comprendre le plan complet
2. Identifie les tâches du sprint demandé
3. Propose un plan d'implémentation détaillé et ATTENDS mon approbation

### Pendant le code :
- Une tâche à la fois, commit après chaque tâche
- Vérifie `npm run build` après chaque modification
- Vérifie visuellement avec `npm run dev` si pertinent
- Messages de commit en français : "fix(sprint-N): description"

### Après le sprint :
- Lance l'audit (`/audit`) pour vérifier les améliorations
- Mets à jour `docs/remediation-plan.md` avec le statut

$ARGUMENTS contient le numéro du sprint (1-3).
